# 3D Genesis Portfolio

A high-end, cinematic 3D portfolio showcasing elite animations, glassmorphism, and interactive experiences.

---

## Live Project URL
🔗 [View Project Here](https://ananthdev.vercel.app)

---

## Tech Stack

- **Next.js** — React framework for production
- **TypeScript** — Strongly typed JavaScript
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Cinematic animations
- **Three.js / React Three Fiber** — 3D graphics
- **shadcn/ui** — Elegant UI components
- **GSAP** — High-performance animations

---

## Quick Start

```sh
# Clone the repository
git clone https://github.com/rolex132/ananth-3d-genesis-folio.git

# Navigate into the project directory
cd ananth-3d-genesis-folio

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

---

## Features

- **Glassmorphism 2.0** — Multi-layered backdrop blurs with holographic color shifts
- **3D Perspective Engine** — Interactive elements that react to mouse movement
- **Cinematic Transitions** — Staggered reveal animations on page load
- **Micro-Interactions** — Haptic-like visual feedback
- **60FPS Performance** — GPU-accelerated animations
