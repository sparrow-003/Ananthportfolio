Featured Projects

DevMatch — Social Platform for Developers

A social networking platform built with React, TypeScript, and Supabase that enables developers to connect, collaborate on projects, and grow their professional network. Features include real-time messaging, profile management, project matching, and activity feeds. The application uses PostgreSQL with Row-Level Security for data protection.

RAG Chat Bot — Document-Grounded AI Assistant

An intelligent chat system using Retrieval-Augmented Generation (RAG) that answers questions based on uploaded documents. Built with Python, LangChain, and OpenAI, this project demonstrates practical LLM integration with vector search, context management, and source citation. Deployed with a React frontend and FastAPI backend.

Naan Mudhalvan — AI & BI Education Platform

An educational platform that empowered 150+ students in Tamil Nadu with hands-on training in Artificial Intelligence and Business Intelligence. Built as a full-stack application with curriculum management, progress tracking, and interactive learning modules. This project reflects my commitment to making technology education accessible in India.
